# turtle
turtle（海龟库）是python的一个绘制图像的函数库，它就像海龟一样，给定指令和坐标，在画布坐标系中一步一步绘制出你想要的图形，比如小猪佩奇、哆啦A梦、皮卡丘等。

python3使用pip install安装时会报错，解决方法参考这篇博客https://blog.csdn.net/m511655654/article/details/86214086
